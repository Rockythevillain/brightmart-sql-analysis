# BrightMart Sales Analysis 🛒

## Project Overview
SQL analysis of BrightMart retail company sales data to identify 
top performing products, customers and cities.

## Business Questions Answered
1. Which product generates the most revenue?
2. Which customer spends the most?
3. Which city generates the most revenue?
4. Which product category performs best?
5. Which month had highest sales?
6. Who are the top 3 customers?
7. Which loyalty tier spends the most?
8. Which cities perform best for Gold/Silver Electronics customers?

## Tools Used
- SQL (SQLite)

## Skills Demonstrated
- JOIN operations (INNER JOIN, LEFT JOIN)
- Aggregate functions (SUM, AVG, COUNT, MAX)
- GROUP BY and HAVING
- COALESCE for NULL handling
- COUNT DISTINCT for unique values
- Complex multi-condition filtering
- Revenue calculation across multiple tables

## Key Findings
- Electronics category generates highest revenue
- Gold tier customers are the highest spenders
- London generates the most revenue of all cities
- January had the highest number of orders
